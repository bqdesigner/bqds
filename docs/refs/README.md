# Refs

Referências visuais pra guiar a criação dos componentes do BQDS.

**Organize por componente:** uma subpasta por item do backlog (§5 do `../AUDIT.md`).

```
refs/
  button/       prints, links, anotações
  card/
  tag/
  ...
```

Prints, screenshots, links (Dribbble, sites), notas soltas — qualquer coisa que ajude a definir o componente antes de codar.
